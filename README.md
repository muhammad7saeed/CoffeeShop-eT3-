
# Coffee Shop Admin Panel

This is a basic Coffee Shop Wep APP using Asp .Net which divide Two Entity (Customer - Drink) that able to admin to Make A Main Function (Create - update - Delete - List) on two Part 

## Backend
- C#
- Entity Frame work (Code First)
- ASP .Net (MVC)

## Front end
- HTML
- CSS
- Java Script
